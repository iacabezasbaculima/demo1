# Demo

## Features

- Build golden container image based on Amazon Linux 2
- Provision ECS cluster with EC2 autoscaling that uses golden container image

## Getting Started

### Pre-requisites

- `terraform`
- `pre-commit`

### Pre-commit hooks

Install pre-commit hooks:

```console
pre-commit install
```

Run pre-commit hooks:

```console
pre-commit run --all-files
```
